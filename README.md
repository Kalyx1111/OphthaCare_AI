## 🚀 Quick Start

### Windows (Recommended)

1. Extract the ZIP anywhere.

2. Double-click:

```text
START_OphthaCare_AI.bat
```

3. Everything installs automatically.

4. Browser opens automatically:

```text
http://localhost:5085
```

5. Accept the disclaimer and begin.

## 🔒 Privacy & Security

### Built-In Security Architecture

- AES-256-GCM encryption
- PBKDF2 key derivation (100,000 iterations)
- Secure local storage (`oc_keys`)
- Provider storage (`ophthacare_provider`)
- API key sanitisation
- Provider whitelist validation
- Rate limiting functionality
- Local-first architecture
- No telemetry or tracking
- Full offline mode supported

API keys remain stored locally and are never transmitted to third-party servers except selected AI providers.

## 🔧 Troubleshooting

### Browser Doesn't Open

Open manually:

```text
http://localhost:5085
```

### Package Installation Fails

Run:

```text
REPAIR_AND_RECOVER.bat
```

### Port Already In Use

Run:

```text
STOP_SERVER.bat
```

### Server Errors

Inspect:

```text
logs\server_*.log
```

or run:

```text
DIAGNOSTIC.bat
```
