# OphthaCare AI v1.0
## Ophthalmology Health Intelligence Platform

## CRITICAL EYE EMERGENCY DISCLAIMER
- **Chemical eye injury:** IRRIGATE IMMEDIATELY with clean water for 20-30 minutes — BEFORE seeking help
- **Penetrating injury:** Do NOT press, do NOT remove foreign body, rigid shield only — Eye Casualty immediately
- **Sudden vision loss, acute angle closure, new flashes/floaters with curtain:** Eye Casualty / **112 India** / 999 UK / 911 US
- All content is AI-generated educational research only — NOT medical advice

## Quick Start (Windows)
1. Extract ZIP to any folder
2. Double-click **START_OphthaCare_AI.bat**
3. Auto-installs everything (2-5 min first time)
4. Browser opens at http://localhost:5085
5. Accept disclaimer and begin

## Security — AES-256-GCM
- API keys encrypted client-side before localStorage storage
- PBKDF2 key derivation (100,000 iterations) from device fingerprint
- Keys never logged by backend server
- Backend rate limiting (30 req/60s), input sanitisation, provider whitelist
